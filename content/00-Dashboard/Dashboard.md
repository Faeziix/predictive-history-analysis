## How to Use

### Learn the Framework
Start at [[predictive-history-moc]] and follow the links. Each note covers one idea and connects to related concepts. Use Obsidian's graph view to see how everything connects. There is no "correct" reading order - follow what interests you.

### Analyze News
1. Run `/analyze-news` in Claude Code to search current news against all claims
2. Run `/analyze-news CLAIM-001` to check a specific claim
3. Review generated notes in this vault
4. Check the Graph View to see connections between claims and evidence
5. Use NotebookLM for deeper AI analysis and deliverables (podcasts, reports)
